## Authentication

All Sendazi API endpoints are secured using **Bearer Token Authentication**.

Include your API token in the `Authorization` header of every request:

```http
Authorization: Bearer YOUR_API_TOKEN

"security": [
  {
    "bearerAuth": []
  }
]
